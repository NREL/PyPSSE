    API interfaces

    The PyPSSE REST API interface enables users to run PyPSSE on a server machine. The interface enables orchestration of multiple simualtion runs in parallel. Future extenions will support

    • User management
    • Result retrival during simulation runs using web socket implementation
    • Ability to upload existing PyPSSE project to the server machine

    psse_service RESTful API documentation 1.0.0

    Create PyPSSE instance, run simulation, get status of running instance, delete running instance etc.

    simulation

    POST /simulators/psse

    Creates a PyPSSE instance

    Request body

    application/json

    {
    "parameters": {
        "filename": "C:/Users/alatif/Desktop/pypsse-code/examples/static_example/Settings/simulation_settings.toml"
    }
}

    Schema of the request body

    {
    "properties": {
        "parameters": {
            "type": "object"
        }
    },
    "type": "object"
}

    Response 200 OK

    application/json

    {
    "Message": "Starting PSSE",
    "Status": "Success,",
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6"
}

    Schema of the response body

    {
    "type": "object"
}

    Response 500 Internal Server Error

    application/json

    {
    "Message": "Failed creating an instance",
    "Status": "Failed,",
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6"
}

    Schema of the response body

    {
    "type": "object"
}

    PUT /simulators/psse

    Run a command on an instance of PyPSSE.

    Request body

    application/json

    {
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6",
    "command": "init",
    "parameters": {}
}

    {
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6",
    "command": "solve_step",
    "parameters": {}
}

    {
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6",
    "command": "get_results",
    "parameters": {
        "params": {
            "Buses": {
                "id_fields": [
                    "PU",
                    "ANGLE"
                ]
            },
            "Loads": {
                "id_fields": [
                    "MVA"
                ]
            }
        }
    }
}

    Schema of the request body

    {
    "properties": {
        "UUID": {
            "example": "96c21e00-cd3c-4943-a914-14451f5f7ab6",
            "format": "UUID",
            "type": "string"
        },
        "command": {
            "example": "open_case",
            "type": "string"
        },
        "parameters": {
            "type": "object"
        }
    },
    "type": "object"
}

    Response 200 OK

    application/json

    {
    "Message": "init command submitted, awaiting response",
    "Status": "Success",
    "UUID": "96c21e00-cd3c-4943-a914-14451f5f7ab6"
}

    Schema of the response body

    {
    "type": "object"
}

    Response 500 Internal Server Error

    application/json

    {
    "code": 500,
    "message": "Error placing command"
}

    Schema of the response body

    {
    "type": "object"
}
